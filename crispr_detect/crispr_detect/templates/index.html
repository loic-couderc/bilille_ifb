Welcome

This appliance provides 2 tools dedicated to bacterial/fungal analysis

  1. Antismash: antiSMASH allows the rapid genome-wide identification, annotation and analysis of secondary metabolite biosynthesis gene clusters in bacterial and fungal genomes
  2. Crispr_detect: Crispr_detect as the name suggests let you disover CRISPRs

This tools can be run with command lines.
Usage:

  1. use a SSH client:ssh -A -p 22 root@<ip>
  2. connect to docker container:docker exec -i -t bilille /bin/bash
  3. run_antismash /upload/<input file> /upload/<output directory> [antismash options]
  4. run_crispr_detect --fasta /upload/<input file> --output_dir /upload/<output directory> [crispr options]
    5. (The /upload/ container dir is mounted on the VM /root/mydisk/bilille/. Please, use this directory to perform your analyses.)
